# Cargo System

**Overview:** Logistics automation with Telegram bot and admin panel.  
**Stack:** PHP, MySQL, Telegram Bot API, Bootstrap, JS.  
**Status:** Production

## Features
- **Core:** Order creation, tracking, notifications
- **Integrations:** Payments, CRM sync, Telegram
- **Admin:** Roles, analytics, shipment statuses
